# Song-Sentiment-Analysis

To run the code it is only necessary to download both the nlp.R file and the csv folder.
Once the code it's open in RStudio set the working directory to the path were the csv is.

Packages required:
-tm
-dplyr
-SnowballC
-ggplot2
-stringr
-tidytext
-wordcloud
-reshape2
-tidyr
-widyr
-igraph
-ggraph
